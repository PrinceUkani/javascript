0 (UNSENT): The XMLHttpRequest object has been created, but the open() method has not been called yet.
1 (OPENED): The open() method has been called, but the send() method has not been called yet.
2 (HEADERS_RECEIVED): The send() method has been called, and the response headers and status are now available.
3 (LOADING): The server has started sending the response data, and partial data is being received. You can access partial data at this state.
4 (DONE): The operation is complete, and the full response has been received from the server. This is when you can process the final data.
